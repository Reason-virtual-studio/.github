# Reason Studios Reason — Portable Virtual Studio with Hardware Emulation

## About Reason

Reason provides a complete virtual studio environment for music producers who want hardware‑style workflow. Reason combines rack view (emulated hardware), virtual instruments (synths, samplers, drum machines), effects (reverb, delay, compression), and mixer. Reason enables users to create music by connecting virtual cables between devices. Reason runs on desktop platforms. Reason is trusted by electronic music producers, beatmakers, sound designers, and hardware enthusiasts worldwide.

## Deep Dive — Virtual Studio Notes

Rack view: scrollable rack, virtual cables, front/back panel editing
Instruments: Europa (wavetable synth), Thor (modular synth), Grain (granular sampler)
Drums: Redrum (drum machine), Kong (drum designer), Beat Map (rhythm generator)
Effects: reverb, delay, compressor, EQ, distortion, chorus, phaser, flanger
Mixer: channel strips, SSL emulation, sends/returns, master bus, dynamics
